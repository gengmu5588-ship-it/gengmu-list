# 轻图社资源站

一个基于纯前端技术栈构建的图集资源展示与交易平台，采用暗色主题设计，支持图集管理、用户系统、支付购买、收藏评论等功能。

## 功能特性

### 前台功能
- **图集浏览** — 按分类筛选图集，支持左右滑动切换动画
- **关键词搜索** — 全站图集搜索，支持标题和描述匹配
- **用户注册/登录** — 支持记住账号密码，换设备需重新登录
- **图集购买** — 支付宝/微信支付码购买，获取网盘下载链接
- **收藏功能** — 收藏图集，同步到个人主页收藏板块
- **评论系统** — 购买用户可对图集进行评价
- **私信站主** — 实时聊天联系站长
- **个人主页** — 展示用户信息、收藏和订单

### 后台管理
- **图集管理** — 发布/编辑/删除图集，支持富文本文案描述
- **分类管理** — 增删改分类，自定义排序
- **订单管理** — 查看所有购买订单
- **用户管理** — 查看注册用户列表及购买记录
- **站主私信** — 与用户实时聊天，红点消息提示
- **首页装扮** — 配置 Logo、横幅、轮播图、版权备案信息
- **收款配置** — 配置支付宝/微信收款码
- **链接检测** — 批量检测图集下载链接状态

## 技术栈

| 技术 | 用途 |
|------|------|

| Tailwind CSS | 工具类 CSS 框架 |

## 快速开始

1. 克隆仓库
   ```bash
   git clone https://github.com/your-username/gallery-site.git
   cd gallery-site
   ```

2. 直接在浏览器中打开 `index.html` 即可运行

   或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000

   # 使用 Node.js
   npx serve .
   ```

3. 访问 `http://localhost:8000`

## 管理后台

- 默认管理员账号：`admin`
- 默认管理员密码：`admin`
- 点击首页右上角"管理"按钮进入后台

## 数据存储

所有数据存储在浏览器 `localStorage` 中，包括：

| Key | 说明 |
|-----|------|
| `albums` | 图集列表 |
| `categories` | 分类列表 |
| `users` | 注册用户 |
| `orders` | 购买订单 |
| `reviews` | 用户评论 |
| `favorites` | 用户收藏 |
| `messages` | 聊天消息 |
| `heroConfig` | 首页装扮配置 |
| `paymentConfig` | 收款配置 |

> ⚠️ 清除浏览器数据将丢失所有信息，建议定期备份。
